## streamrRUN
  
 - Install & Manage a Streamr node
 - Menu driven

  
#### Install (Debian/Ubuntu)

    apt update && apt install -y git curl jq whiptail python3 python3-pip
    git clone https://github.com/logicethos/streamrRUN.git
    cd streamrRUN
    ./streamrRUN


#### Manage your node

    ./streamrRUN
