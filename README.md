## streamrRUN
  
 - Install & Manage a Streamr node
 - Menu driven

  
#### Install (Debian/Ubuntu)

    sudo apt update && sudo apt install -y git curl jq whiptail python3 python3-pip
    git clone https://github.com/logicethos/streamrRUN.git
    cd streamrRUN
    sudo ./streamrRUN


#### Manage your node

    ./streamrRUN
